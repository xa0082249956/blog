| 语法              | 说明                    |
| :-------------: | --------------------- |
| `\`             | 转义                    |
| `.`             | 除`\n`以外的任意字符          |
| `[…]`           | 字符集`…`中的任意一个字符        |
| `[a-c]`         | `a`-`c`中的任意一个字符       |
| `[^…]`          | 除字符集`…`外的任意字符         |
| `\d`            | `[0-9]`               |
| `\D`            | `[^\d]`               |
| `\s`            | `[`<空格>`\t\r\n\f\v]`  |
| `\S`            | `[^\S]`               |
| `\w`            | `[a-zA-Z0-9]`         |
| `\W`            | `[^\w]`               |
| `{m, n}`        | 前一个字符 `m` - `n` 次     |
| `{m}`           | `{m,}`                |
| `*`             | `{0}`                 |
| `+`             | `{1`}                 |
| `?`             | `{0,1}`               |
| `{m, n}?`       | 非贪婪模式                 |
| `^`             | 字符串开头                 |
| `$`             | 字符串末尾                 |
| `\b`            | `\w`与`\W`之间           |
| `\B`            | `\b`                  |
| `|`             | 左右表达式任意一个，先左后右 |
| `(…)`           | 分组，编号1, 2, 3……        |
| `(?P<name>...)` | 分组，除原有编号外指定组名`<name>` |
| `\<number>`     | 引用组`<number>`         |
| `(P=<name>)`    | 引用组`<name>`           |
